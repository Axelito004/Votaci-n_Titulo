## 📖 Descripción del Proyecto

Este repositorio contiene una aplicación web ligera (Single Page Application) desarrollada para gestionar de forma transparente, interactiva y en tiempo real la votación del título definitivo para el proyecto de grado/pasantías. 

El sistema permite a los usuarios (estudiantes, profesores y personal) leer las diferentes propuestas de títulos, emitir un voto de selección única, y dejar recomendaciones opcionales. Cuenta con una interfaz pública para los votantes y un panel de administración privado para auditar los resultados.

---

## ✨ Características Principales

* 📱 **Diseño "Mobile-First" (Responsive):** Interfaz fluida y táctil optimizada para teléfonos móviles, ideal para compartir el enlace vía WhatsApp o Telegram.
* ⚡ **Tiempo Real:** Conexión directa a base de datos en la nube. Los porcentajes y estadísticas se calculan al instante.
* 🎨 **UI/UX Moderna:** Efectos de transición suaves (hover), paleta de colores institucionales (blanco cuarzo y azul) e integración para logos oficiales.
* 🔒 **Panel de Auditoría:** Una vista separada (`admin.html`) que genera un resumen estadístico automático y muestra el registro detallado de cada votante (fecha, opción elegida y recomendaciones).

---

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3 (Vanilla, sin frameworks pesados) y JavaScript.
* **Backend / Base de Datos:** [Supabase](https://supabase.com/) (PostgreSQL como servicio - BaaS).
